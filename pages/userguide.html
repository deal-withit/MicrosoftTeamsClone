User Guide for Microsoft Teams Clone

  • SignUp

    • The user has to visit the SignUp page to enter the credentials for being registered. The Email should be valid and password
    (>8 characters) should be valid.

  • Log In

    • The User has to visit the log in page for entering the credentials and to enter the main page. If user forgot his password then he has to enter the email id and then click the forgot password button. A password reset link will be sent to your registered mail id.

  • Create Call

    • The user has to enter any text at place of unique id and then click Send button and then click on Join call button. You have to share the text with the friend you want to connect with.

  • Join Call

    • You have to enter the shared text (unique id) in the input area and then click on the Join Call button.

  • Chat

    • After visiting the page you have to enter your name and then click the button Join Chat button. Enter the messages and click that send button to send your message in the group.
